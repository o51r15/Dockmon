Notifications

Configure where DockLlama sends alerts when containers are unhealthy, restarted, or enter escalation mode. Uses Apprise — supports 80+ notification services.

Common URL formats

Pushover: pover://user_key@api_token
Discord: discord://webhook_id/webhook_token
Gotify: gotify://hostname/token
Slack: slack://token_a/token_b/token_c/#channel
Telegram: tgram://bot_token/chat_id
Email: mailto://user:pass@gmail.com

See Apprise wiki for all supported services.

Note: Changes apply immediately but are not persisted to config.yaml. Add URLs to your config file to make them permanent across restarts.
